To run this program as a python file: 

please open up terminal in the directory and run the command 

1) pip install -r requirements.txt

2) python Main_Program.py


or simply run the exe file.


Known bugs:

On Mac OS system, after selecting a button (i.e input folder), the program may need to be moved slightly for you to be able to click another button. This is a quirk of the way how the GUI is handled in Mac os and requires a complete rework to fix. The program works as expected.
